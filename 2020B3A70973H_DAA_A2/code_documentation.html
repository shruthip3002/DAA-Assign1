RNA Folding Algorithm Documentation

min_loop_length

min_loop_length: This variable represents the minimum gap required between the ends of each pair in the RNA secondary structure. It follows the "No sharp turns" rule in RNA folding.

pair_check(char a, char b)

pair_check(char a, char b): This function determines whether the given two bases can form a pair in the RNA secondary structure. It returns true if the bases can pair according to the RNA base-pairing rules (A-U, U-A, C-G, or G-C), otherwise false.

OPT(int i, int j, const string &sequence, vector> &DP)

OPT(int i, int j, const string &sequence, vector> &DP): This function computes the values to be filled in the dynamic programming (DP) matrix used in the RNA folding algorithm. It implements the recursive rule of the algorithm to determine the optimal number of base pairs in a given subsequence of the RNA sequence.

traceback(int i, int j, vector> &structure, const vector> &DP, const string &sequence)

traceback(int i, int j, vector> &structure, const vector> &DP, const string &sequence): This function traces back from the upper right corner of the DP matrix to find the pairs in the secondary structure of the RNA sequence. It recursively identifies the base pairs based on the DP matrix values.

write_structure(const string &sequence, const vector> &structure)

write_structure(const string &sequence, const vector> &structure): This function converts the list of pairs in the secondary structure into dot-bracket notation, where each base pair is represented by opening and closing parentheses.

initialize(int N)

initialize(int N): This function initializes the DP matrix with values of -1. It sets the initial values of the DP matrix for subsequences of length less than min_loop_length.

algo(const string &sequence)

algo(const string &sequence): This is the main algorithm function that orchestrates the RNA folding process. It computes the optimal secondary structure of the given RNA sequence using dynamic programming. It returns the dot-bracket notation of the secondary structure and the list of pairs in the secondary structure.

main()

main(): This is the entry point of the program. It reads the RNA sequence from the input file, performs the RNA folding algorithm, writes the dot-bracket notation and pairs of the secondary structure to the output file, and displays the number of pairs and the pairs on the console. It also handles file I/O operations and error checking.
